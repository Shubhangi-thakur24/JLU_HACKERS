# JLU_HACKERS

AI Learning Companion backend built with Django and Django REST Framework.

## Project
- Module 1: Spaced Repetition and Retention Engine
- Features: SM-2 algorithm, Domino Effect parent scheduling, dashboard color coding, and demo day simulation

## Run Backend
1. Open PowerShell in project root.
2. Run:

```powershell
Set-Location "A:\Spaced repetiton statement 1 backend\learning_companion"
& "A:/Spaced repetiton statement 1 backend/.venv/Scripts/python.exe" manage.py migrate
& "A:/Spaced repetiton statement 1 backend/.venv/Scripts/python.exe" manage.py runserver 127.0.0.1:8000
```
